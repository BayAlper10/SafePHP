# SafePHP
A Simple PHP FrameWork.

# Projeye dahil etmek
```php
<?php include("safephp.php"); ?>
```

# Veri tabanına bağlanmak
- Bağlantı kurmak istediğiniz dosyanın üstüne değişkenleri tanımlayın.
```php
<?php
 $sunucu = "localhost";
 $kullanici = "root";
 $parola = "";
 $veritabani = "";
 ?>
```
